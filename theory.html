C Programming Theory

Loops & Conditional Statements

Conditional Statements (`if`, `else if`, `else`)

Conditional statements allow a program to execute certain blocks of code only if a specific condition is true. They are the primary way to make decisions in your code.

The `if-else` Statement

This is the most common conditional statement. If the condition in the parentheses () is true, the code inside the if block is executed. Otherwise, the code inside the else block is executed.

#include <stdio.h> int main() { int age = 20; if (age >= 18) { printf("You are eligible to vote.\\n"); } else { printf("You are not eligible to vote.\\n"); } return 0; }

The `if-else if-else` Ladder

This is used when you need to check multiple conditions. The program checks the conditions from top to bottom and executes the block for the first one that is true. If none are true, the final else block is executed.

#include <stdio.h> int main() { int number = 0; printf("Enter a number: "); scanf("%d", &number); if (number > 0) { printf("The number is positive.\\n"); } else if (number < 0) { printf("The number is negative.\\n"); } else { printf("The number is zero.\\n"); } return 0; }

Looping Statements (`for`, `while`)

Loops are used to execute a block of code repeatedly as long as a certain condition is met. They are essential for automating repetitive tasks.

The `for` Loop

The for loop is ideal when you know exactly how many times you want to repeat the loop. It consists of three parts separated by semicolons:

  1. Initialization: Executes once at the beginning (e.g., int i = 1).
  2. Condition: Checked before each iteration. If false, the loop stops (e.g., i <= 5).
  3. Increment/Decrement: Executes at the end of each iteration (e.g., i++).
#include <stdio.h> int main() { // This loop will run 5 times for (int i = 1; i <= 5; i++) { printf("Iteration number %d\\n", i); } return 0; }

Nested `for` Loops

Placing one loop inside another is called nesting. This is very common for working with 2D structures or creating patterns, where the outer loop controls the rows and the inner loop controls the columns.

#include <stdio.h> int main() { int rows = 4; // Outer loop for rows for (int i = 1; i <= rows; ++i) { // Inner loop for columns for (int j = 1; j <= i; ++j) { printf("* "); } printf("\\n"); // Move to the next line after each row } return 0; }

The `while` Loop

The while loop is used when you don't know the exact number of iterations, and you want the loop to continue as long as a condition is true. The condition is checked *before* each iteration.

#include <stdio.h> int main() { int n = 1; printf("Multiplication table of 5:\\n"); while (n <= 10) { printf("5 * %d = %d\\n", n, 5 * n); n++; // It's crucial to update the loop variable } return 0; }
